Dance Pairs v2
Write a method called combine that takes two integer arrays as parameters — an array of boys' heights and an array of girls' heights (heights given in centimeters). Neither array is guaranteed to be sorted.
Write a program that forms dance pairs (one boy and one girl) according to the following rules:

If the number of boys and girls is equal, pairs are formed so that the tallest boy is paired with the tallest girl, the next tallest with the next tallest, and so on.
If one gender has more members than the other, the tallest extra members of that gender are left without a partner.

The method must return a 2D array with as many elements as there are dance pairs. Each element is a two-element array where the first element is the boy's height and the second is the girl's height.
Write a main method that reads boys' and girls' heights from the command-line arguments, uses combine to form the dance pairs, and prints them neatly. The command-line format is: number of boys, each boy's height separated by spaces, number of girls, each girl's height separated by spaces.
Example
Input:
3 176 179 169 2 172 167
Output:
176 and 172
169 and 167
Note: Command-line arguments (main method parameter) should not be confused with interactive input via System.in.
Hint: An array can be sorted using java.util.Arrays.sort(int[]).
